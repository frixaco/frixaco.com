# Vanilla JS/TS Website

A vanilla JavaScript/TypeScript implementation of my personal website using Bun, Tailwind CSS, and zero frameworks.

## Setup

```bash
bun install
```

## Development

```bash
bun run dev
```

Open http://localhost:3000

## Build

```bash
bun run build
```
